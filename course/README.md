This directory contains a complete teaching course for the Dependency Atlas of Artificial Intelligence.

## Structure

- `lessons/`: Thirteen markdown lesson files, including the course index and twelve layer-oriented modules.
- `instructor_guide.md`: Pacing, facilitation, and classroom use.
- `assessment_rubric.md`: Rubrics for lesson artifacts and capstone grading.
- `capstone.md`: Final project brief and submission requirements.
- `atlas_study_build_graph.md`: Layer-level study graph and cumulative build graph.
- `atlas_build_backlog.md`: Project-oriented implementation checklist.
- `AI_Atlas_Teaching_Deck.pptx`: Slide deck for presentation or classroom use.

## Suggested Path

1. Read `lessons/00-course-index.md`.
2. Complete Lessons 01 through 12 in order.
3. Submit each lesson deliverable.
4. Complete the capstone dependency map and implementation brief.
5. Use `data/atlas.json`, `dist/source_audit_report.md`, and `dist/strict_blocker_report.md` as source-of-truth audit references.

## Completion Standard

A learner has completed the course when they can explain an AI system across at least five atlas layers, defend its data/objective/model/evaluation/deployment choices, and identify evidence required for safe release.
