# AI Dependency Atlas Course

This directory contains teaching and review material for the Dependency Atlas of Artificial Intelligence.

## Start Here

- [Lesson index](lessons/00-course-index.md)
- [Study graph and build graph](atlas_study_build_graph.md)
- [Build backlog](atlas_build_backlog.md)

## Structure

- `lessons/`: Markdown lesson files, one topic per file.
- `atlas_study_build_graph.md`: Layer-level study graph and cumulative build graph.
- `atlas_build_backlog.md`: Project-oriented implementation checklist.
- `AI_Atlas_Teaching_Deck.pptx`: Slide deck for presentation or classroom use.

## Suggested Path

Read the lessons in numeric order, then choose one build project from the backlog as a completion project.
